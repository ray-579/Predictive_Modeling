# Predictive Modeling Using Machine Learning

## Dataset

Iris dataset (built-in from sklearn)

## Algorithms Used

* Logistic Regression
* Decision Tree
* Random Forest

## Steps

* Loaded dataset using sklearn
* Performed train-test split
* Trained models on training data
* Predicted results on test data
* Evaluated using accuracy, confusion matrix, and ROC curve

## Results

* Logistic Regression: ~95–100% accuracy
* Decision Tree: ~90–100% accuracy
* Random Forest: ~95–100% accuracy
